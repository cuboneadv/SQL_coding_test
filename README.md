# SQL_coding_test
SQL 코딩 테스트와 관련된 학습을 정리한 공간입니다.

## 🔧 파일 업로드 자동화
기존 MySQL Workbench를 통해 쿼리 작성 및 다운로드 후 수동으로 업로드하 과정을 Python으로 자동화하습니다. <br>
자세한 내용은 **sql_auto_upload.ipynb** 파일을 참고해 주세요.
